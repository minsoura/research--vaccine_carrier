연구실에서 개발된 백신 캐리어입니다.
http://fab.snu.ac.kr/bbs/board/content.asp?tb=idim_news&page=2&num=394
언론에 보도되기도 하고 신문에도 기사가 나온적이 있습니다.

백신 캐리어는 백신바이알(vial)안에 들어있는 백신의 온도를 유지시키기 위한 미니 냉장고라고 보시면 될것 같습니다.

해당 백신 캐리어는 핸드폰과 연동되어 위치 추적이 가능하고 
백신캐리어의 내부 상태 온도가 웹 사이트에 기록이 됩니다.
웹사이트에서 경로를 확인할 수 있습니다.
대단한 작업은 아니지만 소프트웨어 적인 작업(연동 앱/ 웹사이트 구축)은 연구실에서 할 수 있는 인력이 많이 없어 제가 개발을 담당하였습니다.

해당 백신 캐리어는 탄자니아/ 케냐/말라위 에 보급이 되고 있습니다.
2017년 8월 위치 추적 기능이 탑재된 백신 캐리어가 탄자니아에서 성공적으로 테스팅 되었습니다.
